# Verifier Security Expectations

To maintain the security of the Axelar network, every verifier must:

- Run a full Axelar node.
- Maintain independent chain nodes.
- Monitor ampd uptime to maintain a continuous operational availability for voting.
- Rotate keys on a regular basis.
- Pull rewards regularly.
- Only support chains that are fully trustworthy.
- Be responsive to calls for escalation.
- Participate in regular security drills.
- Adhere to the [color-coded messaging scheme](https://bright-ambert-2bd.notion.site/Partners-Messaging-Scheme-Public-Page-0e5021d084eb4824bdf3216e59d3c39a) between partners and the Interop Labs team.
